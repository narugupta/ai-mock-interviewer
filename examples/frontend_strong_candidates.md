### Turn 1
**Question:**  
Explain how the Virtual DOM works in React.

**Answer:**  
The Virtual DOM is a lightweight JavaScript representation of the real DOM. When state changes, React creates a new virtual DOM tree and compares it with the previous one using a diffing algorithm. It then updates only the changed parts in the real DOM, which improves performance.
